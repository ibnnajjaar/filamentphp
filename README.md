
# Project Yesterday
Enhance your Laravel development with Project Yesterday to meet unrealistic deadlines by delivering it as if it were due yesterday.
All required features are already implemented, tested, and ready to be deployed. These include:
1. User Management
2. Roles and Permissions
3. Activity Log
4. Site Settings

## Installation
Clone the repository and run the following commands:
```bash
composer install
php artisan migrate
php artisan db:seed
php artisan storage:link
npm install
npm run dev
```
